Routes Lupion

Este fue mi proyecto fin de grado superior DAW. Consiste en una aplicación web, realizada con python que permite a los usuarios poder subir sus rutas de ciclismo ya realizadas. Esta aplicación tiene varias secciones. Una sección principal que se encuentra una vez se accede a la aplicación desde el login o si se pulsa en el logo principal, en la que podemos ver las rutas públicas de todos los usuarios registrados en la aplicación. Por otro lado, tenemos una sección titulada Mis rutas, en la que podemos ver y gestionar las rutas de nuestro perfil tanto públicas como privadas. Luego tenemos una sección en la barra de acciones situada justo debajo del título principal, titulada "calcula la dureza" y aquí podremos predecir la dureza que va a tener una ruta rellenando un formulario, introduciendo los datos de esta. También cuenta con una sección que esta situada a la izquierda de esta última, llamada "Vídeos" donde podremos ver listas de youtube obtenidas mediante la api de youtube y podemos descargar en el almacenamiento local el vídeo que más nos interese de esa lista
